# E-Commerce System

A simple e-commerce application built with **Laravel** that allows users to manage products, customers, and orders.  

## Features
- User authentication (Login, Register, Logout)
- Product management: Add, edit, delete products
- Customer management: Add, edit, delete customers
- Order management: Place orders, track total price, update stock
- Bootstrap-based responsive UI
- Sample data preloaded via database seeder

## Tech Stack
- **Backend:** Laravel (PHP)
- **Frontend:** Blade templates, Bootstrap
- **Database:** MySQL 
- **Node.js & npm:** For compiling frontend assets
- **Version Control:** Git & GitHub

## Purpose
This project demonstrates a small-scale full-stack Laravel application suitable for learning Laravel fundamentals, CRUD operations, authentication, and simple e-commerce workflows.
